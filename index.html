CMake 学习笔记

为了写出更加专业规范的工程项目，有必要学习一下CMake工具。这里主要参考Modern CMake,一般指CMake 3.4+ ，甚至是 CMake 3.21+.

本项目主要基于Windows平台进行讲解，并通过VS 2017来进行编译.

相关代码托管在 Github: https://github.com/siyouluo/learn_cmake

环境:

  • Windows 10
  • Visual Studio Community 2017
  • CMake 3.18.5
